== Description ==

El plugin Imreading (EstoyLeyendo) permite visualizar las portadas de los libros que el propietario del blog está leyendo en ese momento y un enlace hacia su dirección en BiblioEteca.

== Installation ==

1. Descargate el .zip e instalalo desde el administrador de wordpress
1. Activa el plugin
1. Arrastra el widget a cualquier barra que tengas configurada
1. Configura el widget con tu nombre de usuario en biblioeteca
